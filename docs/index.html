Features

Everything Explorer should have

Drop-in replacement for the built-in copy and move — with the reliability of a server-grade tool.

⌨️

Keyboard Shortcuts

Ctrl+C, Ctrl+X and Ctrl+V are intercepted in Explorer and powered by robocopy. Delete sends files to the Recycle Bin via the Windows shell API.

📊

Real Progress

Speed, ETA, file count and per-file progress — just like a professional file manager.

⚡

Smart Threading

Auto-detects SSD vs HDD and picks the optimal thread count for maximum throughput.

🔄

Retry on Failure

Configurable retry count and wait — handles locked files, network drops and busy shares.

⚔️

Conflict Resolution

Replace, Keep Both, Skip, or Cancel All — with an "apply to all" option per operation.

🖱️

Right-Click Menu

Quick Copy, Quick Cut and Quick Paste use robocopy for maximum speed. Quick Delete goes straight to the Recycle Bin via the Windows shell — safely and instantly.

🗑️

Recycle Bin Delete

Delete sends files to the Recycle Bin so you can undo mistakes, just as expected.

🔔

System Tray

Runs silently in the background. No floating windows, no dock icons — just works.

Performance

robocopy has always been this fast — now you can use it

The engine ships inside every copy of Windows. RoboExplorer just puts it where it belongs: your right-click menu and Ctrl+C.

Many Small Files — 3,000 files · 3 GB

SSD → SSD  ·  Windows 11  ·  lower is better  ·  community-reported

Windows Explorer ~100 s
TeraCopy ~90 s
RoboExplorer ~40–60% faster ~45 s

robocopy /MT:8 multi-threading — widely reported 30–80% gain on SSD for many small files (SuperUser, r/sysadmin).

Network Share — 10,000 files · 50 GB

1 GbE LAN  ·  Windows 11 → NAS  ·  lower is better  ·  community-reported

Windows Explorer ~18 min
TeraCopy ~16 min
RoboExplorer ~4× faster ~4 min

robocopy /MT:8 keeps the network pipe saturated by issuing parallel reads. 2–5× gains on LAN are widely reported (SuperUser, Microsoft Docs).

Figures represent typical ranges from community benchmarks (SuperUser, r/sysadmin, Microsoft TechNet). Individual results vary by hardware and workload. robocopy is a built-in Windows tool — available since Windows Vista.

Requirements

RoboExplorer is a lightweight tray app with minimal dependencies.

  • .NET 9 Desktop Runtime (free — the installer will prompt if missing)
  • robocopy — included in every edition of Windows since Vista
  • ~10 MB disk space
  • Administrator access for installation (to write to Program Files)
